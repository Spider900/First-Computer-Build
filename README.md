# First-Computer-Build
## Overview 
### One of my first passion projects and the foundation for my interest in computer science was building my own computer in 2020. I didn't have a github account when I built it, so I wasn't able to document my achievement. Now, I realize something this special deserves to be highlighted on this platform as it taught me about computer components in addition to fundamental IT concepts I use daily. I highly encourage others to invest their time in doing the same. The experience gained will be quite valuable along with the money you'll be saving!
